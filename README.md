## To-do List Application

A simple and responsive To-do list application which helps users to manage their daily tasks efficiently.

## Features
 - Add new tasks
 - Deleted completed tasks
 - User-friendly interface

## Technologies used
 - HTML
 - CSS
 - JavaScript

## How to use
 - Enter a task in input field
 - Click the add button to add task to the list
 - Click the delete button to remove a task
